\# Transformer Performance Modeling Framework



End-to-end performance modeling of transformer inference on RISC-V SoCs.



\## Features



\- Parse transformer models from PyTorch, ONNX, TensorFlow

\- Model RISC-V core microarchitecture with vector extensions

\- Estimate latency, throughput, and resource utilization

\- Roofline analysis for performance bounds

\- Optimization exploration (quantization, pruning, fusion)
